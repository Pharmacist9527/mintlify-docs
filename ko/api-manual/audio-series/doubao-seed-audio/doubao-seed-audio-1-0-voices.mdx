Doubao Seed-Audio 1.0 은 [오디오 생성 API](/ko/api-manual/audio-series/doubao-seed-audio/doubao-seed-audio-1-0) 의 `audio_references` 로 보이스를 지정합니다. 각 항목은 다음 중 하나입니다:

- **프리셋 보이스 ID** — 아래 표의 `voice_type` 를 입력합니다(예: `zh_female_vv_uranus_bigtts`)
- **참조 오디오 URL** — 보이스 클로닝용 참조 클립을 업로드합니다

모든 프리셋 보이스는 자연어 프롬프트로 감정·톤·스타일을 제어할 수 있습니다. 중국어 보이스는 영어 텍스트도 읽을 수 있습니다.

<Tip>
  `prompt` 에서 `@音频N` 으로 `audio_references` 의 N 번째 항목을 참조할 수 있으며(번호는 1부터), 한 클립에 여러 보이스를 섞을 수 있습니다. 보이스 이름의 「2.0」 은 보이스 라이브러리 버전을 의미하며 모델 버전과는 무관합니다.
</Tip>
