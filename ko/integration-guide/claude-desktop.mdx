## 개요

![Claude Desktop](/images/integration-guide/claude-desktop-overview.png)

Claude Desktop은 Anthropic의 공식 데스크톱 앱으로, 로컬 환경에서 Claude를 사용할 수 있습니다.

이 가이드는 Claude Desktop을 **EvoLink API**에 연결하고, Anthropic Messages API와 호환되는 모델을 EvoLink를 통해 사용하는 방법을 설명합니다.

**이 가이드에서 다루는 내용:**
- Claude Desktop 설치
- 개발자 모드 활성화
- EvoLink 연결 설정
- 재시작 후 모델 선택

## 사전 준비

시작하기 전에 EvoLink API Key를 준비하세요.

### EvoLink API Key 얻기

- [EvoLink 콘솔](https://evolink.ai/dashboard)에 로그인합니다
- 콘솔에서 API Keys를 찾고 **Create New Key**를 클릭한 뒤 생성된 키를 복사합니다
- API Key는 보통 `sk-`로 시작합니다. 안전하게 보관하세요

## 3단계: 서드파티 추론 설정 열기

Developer 메뉴에서 서드파티 추론 패널을 열고 아래 표에 맞게 입력합니다.

| 항목 | 설명 |
|---|---|
| Gateway base URL | EvoLink 게이트웨이 주소, 권장값: `https://direct.evolink.ai` |
| Gateway API key | `Bearer` 접두사 없이 EvoLink API Key를 그대로 입력합니다 |
| Gateway auth scheme | `bearer` |
| Model list | 권장 사항입니다. Anthropic Messages API를 지원하는 텍스트/채팅 모델 ID만 추가하세요. 아래의 [지원 모델](#supported-claude-models) 표로 바로 이동할 수도 있습니다 |

<Note>
**참고:** 사용 중인 Claude Desktop 버전에서 필드 이름이 약간 다를 수 있습니다. 실제 화면의 레이블을 사용하되 의미는 동일하게 유지하세요.
</Note>

## 4단계: 저장 후 재시작

설정을 저장한 뒤 Claude Desktop을 완전히 종료하고 다시 실행합니다.

<Frame>
![Claude Desktop 재시작 확인 대화상자](/images/integration-guide/claude-desktop-relaunch-dialog.png)
</Frame>

재시작 후 Claude Desktop은 새 서드파티 추론 설정을 사용해야 합니다.

**예상 결과:** 실행 후 사용 가능한 모델 목록이 보이고, 요청이 EvoLink를 통해 전달됩니다.

## 5단계: 모델 선택

재시작한 뒤 모델 선택 영역에서 사용 가능한 모델이 정상적으로 표시되는지 확인하세요.

## 동작 방식

Claude Desktop은 Anthropic 호환 게이트웨이를 통해 EvoLink와 통신합니다.

1. Claude Desktop이 설정한 게이트웨이 주소로 요청을 보냅니다.
2. EvoLink가 요청을 받아 해당 모델 서비스로 전달합니다.
3. 응답이 돌아오면 Claude Desktop은 로컬에서 대화를 계속 표시합니다.

별도의 로컬 프록시는 필요하지 않습니다.

## FAQ

### Developer 메뉴가 보이지 않음

개발자 모드가 활성화되어 있는지, Claude Desktop을 완전히 재시작했는지 확인하세요.

### 설정 후 연결할 수 없음

다음을 확인하세요.

- Base URL이 올바른지
- API Key가 유효한지
- 인증 방식이 `bearer`인지
- Claude Desktop을 완전히 종료했다가 다시 실행했는지

### 모델이 보이지 않음

계정과 네트워크가 정상인지 확인한 뒤, 서드파티 추론 설정이 정상적으로 저장되었는지 확인하세요.

### `Model list`를 꼭 입력해야 하나요?

네, 입력하는 것을 권장합니다. Anthropic Messages API를 지원하는 텍스트/채팅 모델 ID만 추가하세요.

### 모델을 수동으로 지정하려면 무엇을 입력하나요?

위의 "지원 모델" 표에 있는 모델 ID를 Claude Desktop이 기대하는 형식으로 입력하세요.
