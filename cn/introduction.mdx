# EvoLink

企业级 AI 服务网关平台，为开发者提供统一的模型调用入口、任务管理和用量追踪能力。

<CardGroup cols={3}>
  <Card title="统一模型入口" icon="plug">
    通过一套 API 接入图像、视频、文本和音频模型，减少多服务商适配成本。
  </Card>
  <Card title="OpenAI 兼容" icon="code">
    兼容 OpenAI 风格接口和主流 SDK，便于迁移现有应用和工具链。
  </Card>
  <Card title="稳定调度" icon="chart-line">
    支持多渠道调度和容灾切换，降低单一模型或通道不可用带来的影响。
  </Card>
</CardGroup>

## 平台优势

**统一接入**\
一套 API 接入主流图像、视频、文本和音频模型，减少多服务商适配成本

**OpenAI 兼容**\
兼容 OpenAI 标准接口和主流 SDK，便于迁移现有应用

**稳定调度**\
支持多渠道调度和容灾切换，提升模型调用稳定性

**透明计费**\
实时用量统计，灵活的配额管理，每笔消费清晰可追踪

## 服务能力

<CardGroup cols={3}>
  <Card title="图像生成" icon="image" href="/cn/api-manual/image-series/gpt-image-2/gpt-image-2-image-generation">

  <Card title="文本生成" icon="comments" href="/cn/api-manual/language-series/claude/claude-messages-api">

## 核心能力

**智能路由**\
根据模型可用性、响应表现和成本策略进行调度，降低单一通道依赖

**任务管理**\
支持异步任务创建、状态查询和结果获取，适配图像、视频等长耗时任务

**安全访问**\
通过 API Key 管理访问权限，统一管理不同业务和环境的调用凭证

**用量统计**\
提供用量记录和消费追踪，便于团队进行成本核算和配额管理

## 开始使用

<Steps>
  <Step title="获取 API Key">
    在控制台创建并保存密钥：[API Keys](https://evolink.ai/dashboard/keys)
  </Step>
  <Step title="完成第一次调用">
    阅读[快速接入](/cn/quickstart)，使用 GPT Image 2、Seedance 2.0 或 Claude 完成基础调用。
  </Step>
  <Step title="查看模型文档">
    按场景进入[图像 API](/cn/api-manual/image-series/gpt-image-2/gpt-image-2-image-generation)、[视频 API](/cn/api-manual/video-series/seedance2.0/seedance-2.0-overview) 或[文本 API](/cn/api-manual/language-series/claude/claude-messages-api)。
  </Step>
</Steps>
