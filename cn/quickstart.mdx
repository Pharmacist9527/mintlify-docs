三步快速接入EvoLink AI服务平台，开始体验企业级AI能力。

## 准备工作

### 获取API密钥

1. **注册账户**  
   访问 [EvoLink控制台](https://evolink.ai/dashboard) 注册账户

2. **生成密钥**  
   在API管理页面创建新的访问密钥

3. **保存密钥**  
   妥善保存生成的API密钥（格式：`sk-evo-xxxxxxxxxx`）

> **安全提醒**：请将API密钥存储在安全位置，如果怀疑 API Key 泄露，请立即在管理页面重置。

## 接入验证

### 基础调用测试

```bash
curl https://api.evolink.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-5",
    "messages": [
      {
        "role": "user", 
        "content": "Hello"
      }
    ],
    "max_tokens": 100
  }'
```

### 验证响应

成功调用将返回标准的OpenAI格式响应：

```json
{
  "id": "chatcmpl-xxx",
  "object": "chat.completion",
  "model": "gpt-5",
  "choices": [{
    "message": {
      "role": "assistant", 
      "content": "Hello! How can I help you today?"
    }
  }],
  "usage": {
    "total_tokens": 20
  }
}
```

## SDK集成

### Python集成

```python
import openai

# 配置客户端
client = openai.OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.evolink.ai"
)

# 文本生成
response = client.chat.completions.create(
    model="gpt-5",
    messages=[
      {
        "role": "user", 
        "content": "你好"
      }
    ]
)
print(response.choices[0].message.content)

# 图像生成
image = client.images.generate(
    model="gpt-4o-image",
    prompt="professional business meeting"
)
print(image.data[0].url)
```

### Node.js集成

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://api.evolink.ai'
});

// 异步调用示例
async function generateText() {
  const completion = await client.chat.completions.create({
    model: 'gpt-5',
    messages: [
      { 
        role: 'user', 
        content: 'Hello' 
      }
    ]
  });
  
  return completion.choices[0].message.content;
}
```

```python
    # 处理API错误
    pass
```

### 性能优化

**流式响应**  
适用于实时对话场景：

```python
stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "长文本生成"}],
    stream=True
)

for chunk in stream:
    print(chunk.choices[0].delta.content, end="")
```

**批量处理**  
提高大规模处理效率：

```python
# 并发请求处理
import asyncio

async def batch_process(texts):
    tasks = [process_text(text) for text in texts]
    return await asyncio.gather(*tasks)
```
