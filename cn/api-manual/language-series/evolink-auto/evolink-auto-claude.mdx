## 智能模型路由

使用 Anthropic Messages API 格式调用 EvoLink Auto 智能模型路由。

### 核心特点

- **Claude 原生格式**：完全兼容 Anthropic Messages API
- **智能路由**：自动选择合适的模型
- **透明返回**：响应中包含实际使用的模型名称

<Note>
将 `model` 参数设置为 `evolink/auto`，使用 `/v1/messages` 端点。
</Note>

<Note>
**备用地址**：如果您的任务耗时较长，可以将 BaseURL 切换为 `https://direct.evolink.ai`，该地址对长时间任务更加友好。
</Note>
