## 智能模型路由

使用 Anthropic Messages API 格式调用 EvoLink Auto 智能模型路由。

### 核心特点

- **Claude 原生格式**：完全兼容 Anthropic Messages API
- **智能路由**：自动选择合适的模型
- **透明返回**：响应中包含实际使用的模型名称

<Note>
将 `model` 参数设置为 `evolink/auto`，使用 `/v1/messages` 端点。
</Note>

<Note>
**BaseURL 说明**：默认 BaseURL 为 `https://direct.evolink.ai`，对文本模型支持更好，支持长连接；`https://api.evolink.ai` 是多模态主力地址，对文本模型作为备用地址使用。
</Note>
