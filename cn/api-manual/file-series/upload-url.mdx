## 使用流程

当你的源文件已经存在于公网 URL，并希望把它登记成后续模型调用可直接复用的文件地址时，可以使用这个接口。

1. 提交远程文件 URL。
2. 将返回的 `file_url` 或 `download_url` 填入 `image_urls`、`audio_url`、`video_urls` 或 Gemini 的 `fileUri` 等参数。
3. 提交 AI 请求后，使用返回的 `task_id` 轮询任务直到完成。
