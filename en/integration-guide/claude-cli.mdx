## Overview

![Claude CLI Interface](/images/integration-guide/image-9.png)

Claude CLI is the official command-line tool from Anthropic for interacting with Claude models in the terminal.
By integrating Claude CLI with **EvoLink API**, you can directly access Claude model capabilities through EvoLink.

## Prerequisites

Before configuring, make sure you have:

### 1. Install Node.js and npm

- Download and install from [Node.js official website](https://nodejs.org/)
- Recommended: **Node.js v18 or higher**
- Verify installation:

```bash
node -v
npm -v
```

### 2. Get EvoLink API Key

- Log in to [EvoLink Dashboard](https://evolink.ai/dashboard)
- Create and copy your API Key
- API Key usually starts with `sk-`

## Step 1: Install Claude CLI

### 1. Global Installation

```bash
npm install -g @anthropic-ai/claude-code
```

### 2. Verify Installation

```bash
claude --version
```

![Claude CLI Version](/images/integration-guide/image-8.png)

## Step 2: Configure EvoLink API

Claude CLI supports multiple configuration methods. Choose the one that works best for you.

### Method 1: Configure settings.json (Recommended)

This is the recommended approach with permanent effect.

**Config file location:**

<Tabs>
  <Tab title="Windows">
    ```
    C:\Users\{username}\.claude\settings.json
    ```
  </Tab>
  <Tab title="macOS">
    ```
    ~/.claude/settings.json
    ```
  </Tab>
  <Tab title="Linux">
    ```
    ~/.claude/settings.json
    ```
  </Tab>
</Tabs>

**Edit the config file with the following content:**

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "your-evolink-api-key",
    "ANTHROPIC_BASE_URL": "https://api.evolink.ai",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1"
  },
  "permissions": {
    "allow": [],
    "deny": []
  },
  "model": "claude-sonnet-4-5-20250929"
}
```

### Method 2: Temporary Environment Variables (Current Terminal Only)

Suitable for quick testing or temporary configuration. Settings are lost when the terminal is closed.

<Tabs>
  <Tab title="PowerShell">
    ```powershell
    $env:ANTHROPIC_BASE_URL="https://api.evolink.ai"
    $env:ANTHROPIC_AUTH_TOKEN="your-evolink-api-key"
    ```
  </Tab>
  <Tab title="CMD">
    ```cmd
    set ANTHROPIC_BASE_URL=https://api.evolink.ai
    set ANTHROPIC_AUTH_TOKEN=your-evolink-api-key
    ```
  </Tab>
  <Tab title="macOS / Linux">
    ```bash
    export ANTHROPIC_BASE_URL="https://api.evolink.ai"
    export ANTHROPIC_AUTH_TOKEN="your-evolink-api-key"
    ```
  </Tab>
</Tabs>

### Method 3: Permanent Environment Variables

If you prefer to configure environment variables at the system level:

<Tabs>
  <Tab title="Windows GUI">
    1. Right-click "This PC" → "Properties" → "Advanced system settings" → "Environment Variables"
    2. Under "User variables", create new entries:
       - Variable name: `ANTHROPIC_BASE_URL`, Value: `https://api.evolink.ai`
       - Variable name: `ANTHROPIC_AUTH_TOKEN`, Value: your API Key
    3. Click OK and restart the terminal
  </Tab>
  <Tab title="PowerShell Permanent">
    ```powershell
    [System.Environment]::SetEnvironmentVariable('ANTHROPIC_BASE_URL', 'https://api.evolink.ai', 'User')
    [System.Environment]::SetEnvironmentVariable('ANTHROPIC_AUTH_TOKEN', 'your-evolink-api-key', 'User')
    ```
    Restart the terminal for changes to take effect.
  </Tab>
  <Tab title="macOS / Linux">
    Edit your shell config file (Bash: `~/.bashrc` or `~/.bash_profile`, Zsh: `~/.zshrc`) and add:

    ```bash
    export ANTHROPIC_BASE_URL="https://api.evolink.ai"
    export ANTHROPIC_AUTH_TOKEN="your-evolink-api-key"
    ```

    Apply the changes:

    ```bash
    source ~/.bashrc  # or source ~/.zshrc
    ```
  </Tab>
</Tabs>

### Verify Configuration

After configuration, verify that environment variables are set correctly:

<Tabs>
  <Tab title="PowerShell">
    ```powershell
    echo $env:ANTHROPIC_BASE_URL
    echo $env:ANTHROPIC_AUTH_TOKEN
    ```
  </Tab>
  <Tab title="CMD">
    ```cmd
    echo %ANTHROPIC_BASE_URL%
    echo %ANTHROPIC_AUTH_TOKEN%
    ```
  </Tab>
  <Tab title="macOS / Linux">
    ```bash
    echo $ANTHROPIC_BASE_URL
    echo $ANTHROPIC_AUTH_TOKEN
    ```
  </Tab>
</Tabs>

## Step 3: Start Using Claude CLI

### 1. Enter a safe working directory

```powershell
cd D:\mycode\ClaudeTest
```

### 2. Interactive mode

```shell
claude
```

### 3. Single query mode

```shell
claude "Who are you"
```

If you receive a normal response, the configuration is successful.

## FAQ

### 1. How do I switch models?
Edit the `model` field in `settings.json`.

### 2. How do I verify EvoLink is working?
Run `claude "Who are you"`. A normal response indicates success.

### 3. What if I get 401/403 errors?
Check that `ANTHROPIC_AUTH_TOKEN` is set correctly and `ANTHROPIC_BASE_URL` is `https://api.evolink.ai`.

### 4. How do I upload images?
- Option 1: Reference the image path
- Option 2: Drag and drop an image into the terminal
- Option 3: Paste an image directly

## Notes

<Warning>
Run Claude CLI in a safe directory. It can only read files in the current working directory.
</Warning>

<Tip>
If you previously logged in with an official account, clear the `ANTHROPIC_AUTH_TOKEN` environment variable or override it in `settings.json`.
</Tip>
