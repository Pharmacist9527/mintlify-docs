title: "Gemini CLI"
description: "Connect Gemini CLI to EvoLink.AI"
---

## Prerequisites

- Node.js v18 or higher
- EvoLink.AI API Key

## Step 1: Install Gemini CLI

```bash
npm install -g @google/gemini-cli
```

Verify installation:

```bash
gemini --version
```

## Step 2: Find Installation Directory

```bash
npm root -g
```

Example output: `D:\develop\NodeJS\node_modules`

## Step 3: Modify API Endpoint

Modify 2 files:

**File 1:** `{install_dir}/@google/gemini-cli/node_modules/@google/genai/dist/node/index.mjs`

Find line **~11222**:

```javascript
// Before
initHttpOptions.baseUrl = `https://generativelanguage.googleapis.com/`;

// After
initHttpOptions.baseUrl = `https://api.evolink.ai/`;
```

**File 2:** `{install_dir}/@google/gemini-cli/node_modules/@google/genai/dist/node/index.cjs`

Find line **~11244**, make the same change.

## Step 4: Modify Default Model

**File:** `{install_dir}/@google/gemini-cli/node_modules/@google/gemini-cli-core/dist/src/config/defaultModelConfigs.js`

Find line **~96-99**:

```javascript
// Before
classifier: {
    extends: 'base',
    modelConfig: {
        model: 'gemini-2.5-flash-lite',

// After
classifier: {
    extends: 'base',
    modelConfig: {
        model: 'gemini-3-flash-preview',
```

## Step 5: Configure API Key

**Option 1: Temporary (current session only)**

```powershell
$env:GEMINI_API_KEY = "your-evolink-api-key"
```

**Option 2: Permanent**

```powershell
[Environment]::SetEnvironmentVariable("GEMINI_API_KEY", "your-evolink-api-key", "User")
```

Restart terminal after setting.

## Step 6: Verify Connection

```bash
cd ~
gemini "Hello"
```

If you receive a normal response, the configuration is successful.

## Notes

<Warning>
Updates will overwrite modifications. After running `npm update -g @google/gemini-cli`, you need to modify the files again.
</Warning>

<Tip>
Avoid running in the system32 directory to prevent permission warnings.
</Tip>
