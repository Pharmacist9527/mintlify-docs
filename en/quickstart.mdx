# Quick Integration Guide

Three steps to quickly integrate EvoLink AI service platform and start experiencing enterprise-level AI capabilities.

---

## Preparation

### Get API Key

1. **Register Account**  
   Visit [EvoLinkConsole](https://evolink.ai/dashboard) Register Account

2. **Generate Key**  
   Create a new access key in the API management page

3. **Save Key**  
   Safely save the generated API key (format: `sk-evo-xxxxxxxxxx`)

> **Security Reminder**:Please store your API key in a secure location. If you suspect your API Key has been compromised, reset it immediately in the management page.

## Integration Verification

### Basic Call Test

```bash
curl https://api.evolink.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-5",
    "messages": [
      {
        "role": "user", 
        "content": "Hello"
      }
    ],
    "max_tokens": 100
  }'
```

### Verify Response

Successful calls will return standard OpenAI format responses:

```json
{
  "id": "chatcmpl-xxx",
  "object": "chat.completion",
  "model": "gpt-5",
  "choices": [{
    "message": {
      "role": "assistant", 
      "content": "Hello! How can I help you today?"
    }
  }],
  "usage": {
    "total_tokens": 20
  }
}
```

## SDK Integration

### Python Integration

```python
import openai

# Configure Client
client = openai.OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.evolink.ai"
)

# Text Generation
response = client.chat.completions.create(
    model="gpt-5",
    messages=[
      {
        "role": "user", 
        "content": "Hello"
      }
    ]
)
print(response.choices[0].message.content)

# Image Generation
image = client.images.generate(
    model="gpt-4o-image",
    prompt="professional business meeting"
)
print(image.data[0].url)
```

### Node.js Integration

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://api.evolink.ai'
});

// Async Call Example
async function generateText() {
  const completion = await client.chat.completions.create({
    model: 'gpt-5',
    messages: [
      { 
        role: 'user', 
        content: 'Hello' 
      }
    ]
  });
  
  return completion.choices[0].message.content;
}
```

## Production Deployment

### Error Handling

```python
try:
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": "Hello"}]
    )
except openai.RateLimitError:
    # Handle rate limiting
    pass
except openai.APIError as e:
    # Handle API errors
    pass
```

### Performance Optimization

**Streaming Response**  
Suitable for real-time conversation scenarios:

```python
stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Long Text Generation"}],
    stream=True
)

for chunk in stream:
    print(chunk.choices[0].delta.content, end="")
```

**Batch Processing**  
Improve large-scale processing efficiency:

```python
# Concurrent request processing
import asyncio

async def batch_process(texts):
    tasks = [process_text(text) for text in texts]
    return await asyncio.gather(*tasks)
```
