## Related Workflow

Use this endpoint after you submit an async AI request. The usual workflow is:

1. Upload local files first when the target API expects hosted URLs.
2. Submit the AI request and capture the returned `task_id`.
3. Query this endpoint until the task reaches a terminal status, then read the final output.
