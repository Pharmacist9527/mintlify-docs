## Workflow

Use this endpoint when your source asset already exists at a public URL and you want EvoLink to register it for later model calls.

1. Submit the remote file URL.
2. Reuse the returned `file_url` or `download_url` in parameters such as `image_urls`, `audio_url`, `video_urls`, or Gemini `fileUri`.
3. Submit the AI request and poll the resulting `task_id` until the task is completed.
