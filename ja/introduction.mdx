# EvoLink

EvoLink は、統一されたモデルアクセス、タスク管理、利用状況の追跡を必要とする開発者向けのエンタープライズ AI ゲートウェイプラットフォームです。

<CardGroup cols={3}>
  <Card title="統一モデルアクセス" icon="plug">
    1つの API から画像、動画、テキスト、音声モデルにアクセスし、複数プロバイダーの連携コストを削減します。
  </Card>
  <Card title="OpenAI 互換" icon="code">
    OpenAI 形式の API と主要 SDK に対応し、既存アプリケーションやツールチェーンの移行を容易にします。
  </Card>
  <Card title="安定したルーティング" icon="chart-line">
    マルチチャネルルーティングとフェイルオーバーにより、単一モデルや上流チャネルの障害による影響を抑えます。
  </Card>
</CardGroup>

## プラットフォームの利点

**統一アクセス**\
1つの API で主要な画像、動画、テキスト、音声モデルに接続し、複数プロバイダーへの適配コストを削減します

**OpenAI 互換**\
OpenAI 形式のインターフェースと主要 SDK に対応し、既存アプリケーションの移行を簡単にします

**安定したルーティング**\
マルチチャネルルーティングとフェイルオーバーにより、モデル呼び出しの安定性を高めます

**透明な課金**\
利用量をリアルタイムで確認し、クォータを柔軟に管理し、各消費を明確に追跡できます

## コア機能

**インテリジェントルーティング**\
モデルの可用性、応答状況、コスト方針に基づいてリクエストを振り分け、単一チャネルへの依存を減らします

**タスク管理**\
非同期タスクの作成、ステータス照会、結果取得により、画像や動画など時間のかかる処理に対応します

**安全なアクセス**\
API Key によって、業務、環境、チームごとのアクセス権限を管理できます

**利用状況の追跡**\
利用量と消費記録を追跡し、コスト配分とクォータ管理に活用できます
