<Tip>
**ストリーミング**：URL の `generateContent` を `streamGenerateContent` に置き換えると、ストリーミングレスポンスが有効になり、コンテンツをリアルタイムでチャンクごとに受信できます。
</Tip>

<Note>
**代替URL**：長時間かかるタスクの場合、BaseURL を `https://direct.evolink.ai` に切り替えることができます。長時間タスクに最適化されています。
</Note>
