Doubao Seed-Audio 1.0 は[音声生成 API](/ja/api-manual/audio-series/doubao-seed-audio/doubao-seed-audio-1-0) の `audio_references` でボイスを指定します。各項目は次のいずれかです：

- **プリセットボイス ID** — 下の表の `voice_type` を入力します（例：`zh_female_vv_uranus_bigtts`）
- **参照音声 URL** — ボイスクローン用の参照クリップをアップロードします

すべてのプリセットボイスは、自然言語のプロンプトで感情・トーン・スタイルを制御できます。中国語ボイスは英語テキストの読み上げにも対応します。

<Tip>
  `prompt` 内で `@音声N` を使うと `audio_references` の N 番目の項目を参照でき（番号は 1 から）、1 つのクリップで複数のボイスを混在させられます。ボイス名の「2.0」はボイスライブラリのバージョンを表し、モデルのバージョンとは関係ありません。
</Tip>
